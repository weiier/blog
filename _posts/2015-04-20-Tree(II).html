We continue our discussion about tree.

In this time,we use java programming language and the binary tree node defined as follows:

{% highlight java %} /** * Definition for binary tree * public class TreeNode { * int val; * TreeNode left; * TreeNode right; * TreeNode(int x) { val = x; } * } */ {% endhighlight%}

No matter what languages we used, the algorithms are the same.Of course, recursive solution is trivial, when we have time, we could do it iteratively.

Solution for the following questions:to travel a tree in recursive way, when to visit the "root node" is the key point and it decide the travel order is preorder,inorder or postorder.

Binary Tree Preorder Traversal

Given a binary tree, return the preorder traversal of its nodes' values.

{% highlight java %} public class Solution { public List preorderTraversal(TreeNode root) { List result = new ArrayList(); if(root == null) return result; else { List resultL = preorderTraversal(root.left); List resultR = preorderTraversal(root.right); result.add(root.val); result.addAll(resultL); result.addAll(resultR); } return result; } } {% endhighlight%}
